# 🚀 DevOps Project: Flask App on Kubernetes (EKS)

## 📌 Overview
This project demonstrates a complete DevOps pipeline by deploying a Flask + MySQL application using Docker, Kubernetes (EKS), and CI/CD.

---

## 🛠 Tech Stack
- AWS (EKS, EC2, LoadBalancer)
- Docker
- Kubernetes
- GitHub Actions (CI/CD)
- Prometheus & Grafana (Monitoring)

---

## ⚙️ Features
- Containerized Flask application
- Kubernetes deployment with scaling
- Secure secrets management
- LoadBalancer + Ingress setup
- CI/CD automation
- Monitoring with Grafana

---

## 🚀 Deployment Flow
GitHub → Docker → Kubernetes → AWS LoadBalancer

---

## 📊 Monitoring
Grafana dashboard used for:
- CPU usage
- Memory usage
- Pod health

---

## 🔥 Key Learnings
- Debugged ImagePullBackOff issues
- Configured Kubernetes Services & Ingress
- Implemented CI/CD pipeline
- Deployed production-like architecture
